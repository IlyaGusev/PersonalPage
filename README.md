# PersonalPage
Личный сайт на Django


Install Docker:

>> sudo sh -c "echo deb https://get.docker.com/ubuntu docker main > /etc/apt/sources.list.d/docker.list"

>> sudo apt-get update

>> sudo apt-get install lxc-docker


Build:

>> sudo sh build.sh 


Connect through Bash:
>> sudo docker exec -it $JOB bash
